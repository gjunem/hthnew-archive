# hthnew-archive

本仓库用于归档和发布多个独立 HTML 页面。页面内容涵盖不同主题，彼此无直接关联，仅为独立的静态文件集合。

## 项目简介

hthnew-archive 是一个面向通用 HTML 页面存档的仓库。各页面均以独立的 HTML 文件形式存放，不依赖特定域名或网站。适合需要长期保存、便于分发或离线查看的 HTML 文档。

## 目录说明

仓库根目录下存放所有 HTML 文件。文件命名采用简洁方式，便于识别和引用。目录结构如下：

```
/
├── index.html          # 可选：页面导航或列表
├── page-example.html   # 示例页面
└── ...
```

如有子目录，会在对应文件夹中额外说明。

## 页面归档说明

- 每个 HTML 页面均为独立文件，不依赖外部资源（除非在页面内显式声明）。
- 归档页面可能包含静态内容，也可能包含简单交互（如 JavaScript 脚本）。请自行评估使用风险。
- 页面内容不代表本仓库立场，仅为存档目的而收录。
